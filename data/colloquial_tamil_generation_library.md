## For LLMs generating natural spoken Tamil
### Built from: *Colloquial Tamil* — Asher & Annamalai (Routledge) + authentic usage

## HOW TO USE THIS FILE

This is a **generation library**, not a dictionary.  
LLMs must copy the **rhythm, structure, and feel** of examples — not just swap words.  
Natural spoken Tamil lives in particles, trailing sentences, incomplete thoughts, and code-switching flow.

## PART 1: THE MOST IMPORTANT RULE — SPOKEN ≠ WRITTEN

The book says it clearly: *"The language of writing differs considerably from the language of everyday conversation."*  
Colloquial forms are almost never written in Tamil script in everyday life.

| Written / Formal (NEVER use) | Spoken / Colloquial (ALWAYS use) |
|---|---|
| அவர்கள் | அவங்க |
| இல்லை | இல்ல |
| என்னவென்று | என்னன்னு |
| போகிறேன் | போறேன் |
| செய்கிறேன் | பண்றேன் / செய்றேன் |
| வருகிறேன் | வர்றேன் |
| நன்றி | thanks / சரி (நன்றி is almost never said) |
| சந்தோஷமாக | happy-ஆ / சந்தோஷமா |
| கவலைப்படுகிறேன் | கவலையா இருக்கு |
| மிகவும் நன்றி | romba thanks (book confirms: formal Tamil uses *mikka nanri*, spoken uses *romba thanks*) |
| நான் சென்றேன் | நான் போனேன் / போயிட்டேன் |
| அவர் வந்தார் | அவரு வந்தாரு / அவங்க வந்தாங்க |

## PART 2: CORE GRAMMATICAL PARTICLES
### (These make or break naturalness — use constantly)

### -டா / -டி (da / di)
The single most important marker of casual intimate speech. Book confirms this is pervasive in colloquial Tamil.
- `-டா` = to male / neutral
- `-டி` = to female
- Goes at end of any statement to make it casual: `என்னடா` `போடா` `வாடா` `சரிடா`
- **WRONG:** "அது என்னவென்று தெரியவில்லை" | **RIGHT:** "அது என்னடா தெரியல"

### -ங்க (-nga)
Marks respect/politeness without formality. Added to verbs when speaking UP to someone.
- `சொல்லுங்க` (please tell) / `வாங்க` (please come) / `பாருங்க` (please see)
- Book uses this extensively in all polite dialogue

### -லே / -லே (-le)
Casual locative. More natural than -இல்
- `கல்லூரிலே` not `கல்லூரியில்` | `வீட்டிலே` not `வீட்டில்`

### -தான் (-daan / -taan)
Emphasis. "It IS / THAT'S the thing"
- `நான்தான்` = it IS me | `அதுதான்` = THAT'S it, exactly | `நாளைக்குத்தான்` = it's specifically tomorrow

### -ஆ (-aa)
Question marker AND disbelief marker.
- Question: `வந்தாரா?` = Did he come?
- Disbelief: `அப்படியா?` = Really?? / Is that so?
- Book: `aamaa` (ஆமா) = yes | `ille` (இல்ல) = no — these are the core yes/no of colloquial Tamil

### -மோ / -ஓ (-mo / -o)
Uncertainty, softening, wondering.
- `வருவாங்களோ?` = I wonder if they'll come
- `என்னமோ...` = who knows / I don't even know (resignation)

### -வே / -யே (-ve / -ye)
Emphasis particle — "even / itself / at all"
- `வரவே மாட்டாங்க` = won't come AT ALL
- `அதுவே சரி` = THAT itself is right
- `பார்க்கவே இல்ல` = didn't even look

### -லயே (-layee)
"Right there / in that very" — strong emphasis on location
- `கல்லூரிலயே பாத்தேன்` = saw it right there at college itself

### -ஆச்சு / -ஆச்சே (aaccu / aaccee)
"It's done / happened / become" — completion with emotion
- Book dialogue: `saivam aaccee` = "I've become vegetarian" (with surprise/resignation)
- `ஆகிவிட்டது` in written → `ஆயிடுச்சு / ஆச்சு` in spoken
- `viΩaa aaccee` = the festival has already happened

## PART 4: SENTENCE RHYTHM PATTERNS

### How natural Tamil sentences START
Natural Tamil puts emotional reaction FIRST, then content.

| WRONG (content first) | RIGHT (emotion first) |
|---|---|
| பெட்ரோல் விலை ஏறிவிட்டது — வருத்தமாக இருக்கிறது | ஐயய்யோ — petrol price எங்கயோ போச்சே... |
| விஜய் CM ஆனார் என்று நம்பமுடியவில்லை | என்னடா இது — Vijay CM-ஆ? நம்பவே முடியல... |

**Common openers:**
- `என்னடா இது —` (exasperation)
- `அடப்பாவி,` (mild outrage)
- `ஐயய்யோ,` (distress)
- `ஹா —` (laughing)
- `ஆமா-ஆ?` (disbelief)
- `சீரியஸ்லி —` (emphasis)
- `பாரு —` (attention-getting)
- `அதுதான் —` (agreement)
- `நான் சொன்னேன்லே —` (I told you)
- `a∂a` (அட — mild surprise, book uses: `a∂a, niinga eppa Indiyaavukku vandiinga?`)

### How natural Tamil sentences END
Sentences trail off, question, or hang. They don't end cleanly like written Tamil.

**Trailing off:**
- `...என்னமோ` = who knows / I give up
- `...போச்சு` = it's gone / that's that
- `...தெரியல` = I don't know
- `...விடு` = just leave it
- `...என்ன பண்றோம்` = what do we do
- `...சரியில்ல` = something's off

**Tag questions (from book):**
- `இல்லையா?` = isn't it? / right?
- `சரியா சொன்னேன்லே?` = I told you, didn't I?
- `தெரியுமா?` = do you know? / you know right?
- `ஆச்சரியமா இல்ல?` = isn't that surprising?
- `mu∂iyumaa?` = is it possible? / can you?

**Trailing into silence (for TTS — dots = breath gaps):**
- `அப்புறம்.........` = and then......... (let it hang)
- `ஆனா.........` = but.........
- `seriously-யா.........` = the magic pattern — long pause sounds most human

### Reduplication = Intensity (from book grammar section)
The book explicitly covers this: repeating a word intensifies meaning.
- `romba romba` = VERY very much
- `பாருங்க பாருங்க` = look look! (urgent)
- `வந்து வந்து` = came again and again
- `சொல்லி சொல்லி பாத்தான்` = tried telling him again and again (book: `solli solli paattaan`)
- `ஆமா ஆமா` = enthusiastic yes
- `போடா போடா` = emphatic dismissal
- `ஐயய்யோ ஐயய்யோ` = extreme distress

## PART 5: CODE-SWITCHING PATTERNS (Tamil + English)

The book notes: *"Modern Tamil, especially the spoken variety, makes use of a number of English words."*

### Grammar rule for English verbs: add -பண்ணு or -ஆகு
| English | Tamil colloquial form |
|---|---|
| try | try-பண்ணு |
| ready | ready-ஆச்சு |
| on | on-பண்ணு |
| miss | miss-பண்ணு |
| plan | plan-பண்றோம் |
| manage | manage-ஆகுது / manage-பண்றேன் |
| pass | pass-பண்ணேன் (book: `paas pa==unaan`) |

### Natural mid-sentence English words
These flow naturally — don't force Tamil equivalents:
`seriously, literally, actually, honestly, basically, technically, totally, random, dramatic, awkward, valid, toxic, cringe, waste, scene, logic, level, update`

### Full natural code-switch patterns:
- `seriously-யா இது?` = English word + Tamil question marker
- `HOD கூப்பிட்டாங்க, literally` = Tamil sentence + English emphasis at end
- `அவங்க so dramatic-ஆ பண்றாங்க` = Tamil frame + English adjective
- `enna logic-டா இது` = Tamil exclamation + English noun + Tamil particle
- `government-க்கு என்னாச்சு` = English noun + Tamil dative suffix
- `fully waste-ஆ போச்சு` = English + Tamil completive
- `ngl அது romba awkward-ஆ இருந்துது` = Modern slang entry + Tamil

## PART 8: SITUATIONAL REACTION LIBRARY
### (Complete authentic responses — copy the rhythm, not just the words)

### Price rise / cost of living
**WRONG:** `விலை அதிகமாகிவிட்டது என்று தெரிகிறது`  
**RIGHT:** `எங்கயோ போச்சு — petrol எண்பது ரூபாய்னு... என்ன பண்றோம் சொல்லுடா`  
**RIGHT:** `அடப்பாவி, onion price பாத்தியா? அம்மா market போனா shock-ஆயிட்டு வருவாங்க...`

### Something unbelievable
**WRONG:** `இதை நம்ப முடியவில்லை`  
**RIGHT:** `என்னடா — சீரியஸ்லியா சொல்ற? நம்பவே முடியல...`  
**RIGHT:** `அப்படியே நம்பிட்டே — யாரு சொன்னான்னு கேக்கலயா?`

### College / exam stress
**WRONG:** `தேர்வு நெருங்கிவிட்டது, படிக்க வேண்டும்`  
**RIGHT:** `internal நாளைக்கு, studied-ஆ? நான்... partial-ஆ பாத்தேன்... manage ஆகும் பாரு`  
**RIGHT:** `attendance 75% இல்லன்னு HOD கூப்பிட்டாங்க......... என்ன பண்றோம்`

### Someone spreading gossip
**WRONG:** `அவர்கள் தேவையில்லாமல் பேசுகிறார்கள்`  
**RIGHT:** `சும்மா இருக்க மாட்டாங்க — இவங்களுக்கு என்ன வேலை இல்லையா seriously`  
**RIGHT:** `பேச்சே வராது... அவங்க யாரு சொன்னாங்கன்னு கேட்டியா?`

### Bad cinema / movie
**WRONG:** `படத்தின் கதை சரியாக இல்லை`  
**RIGHT:** `ஹா — Vijay movie story என்னான்னு கேட்டா... சூப்பரா இருக்குன்னு மட்டும் தான்...`  
**RIGHT:** `waste of money-டா, ஒரே ஒரு scene கூட brain-ல போகல`

### Politician / authority figure failing
**WRONG:** `அரசியல்வாதிகள் பொய் சொல்கிறார்கள்`  
**RIGHT:** `அவங்க கண்ணுல தண்ணியே இல்ல... சொல்றதே சொல்றாங்க, ஆனா நடக்கவே நடக்காது`  
**RIGHT:** `கொஞ்சம் யோசிச்சு பேசுன்னு சொல்ல ஆளில்லையா — அடப்பாவி`

### Power cut (universal frustration — directly from book)
**FROM BOOK (authentic dialogue):** `இன்னைக்கு பகல் பூராவும் power cut. கோபம் கோபமா வந்துடு. என்ன செய்றது?`  
This line is authentic textbook Tamil — use as a template.

### Someone being dramatic / overdoing it
**WRONG:** `அவர் மிகவும் அதிகமாக நடிக்கிறார்`  
**RIGHT:** `அவங்க so dramatic-ஆ பண்றாங்க seriously, ஒரு சின்ன விஷயத்துக்கு இவ்வளவு scene-ஆ`

### Agreeing strongly with someone
**WRONG:** `ஆம், நீங்கள் சரியாக சொல்கிறீர்கள்`  
**RIGHT:** `சரியா சொன்னே — அதுதான் நான் சொல்ல வந்தது! நாம் சொன்னா கேக்கவே மாட்டாங்க...`

### Something went wrong / failed
**WRONG:** `அது சரியாக நடக்கவில்லை`  
**RIGHT:** `ஆகல, ஆகவே ஆகல — நான் சொன்னேன்லே, கொஞ்சம் யோசிச்சு பண்ணுன்னு... என்னமோ போ........`

### Hesitation / not wanting to do something
**FROM BOOK:** `avangee tayangu` = hesitate  
**RIGHT:** `சொல்ல... தெரியல, ஆமா... என்னமோ... நீயே பாரு`

## PART 10: PUNCTUATION → PROSODY MAP
### (For TTS: what punctuation creates what sound effect)

| Punctuation | Sound effect |
|---|---|
| `,` | Brief pause, mid-thought breath |
| `...` | Trailing off, thinking, uncertain pause |
| `..........` | Long trailing silence, resignation, disbelief |
| `—` | Abrupt stop before next thought (double-take effect) |
| `!` | Outrage, excitement, emphasis |
| `?` | Question, or disbelief with rising intonation |
| `seriously-யா..........` | THE KEY PATTERN — long breath gap = most human sounding |

**TTS pacing by emotion (Sarvam pace parameter):**
- `[shocked]` → pace 1.1 (fast, clipped)
- `[laughing]` → pace 0.95 (light)
- `[outraged]` → pace 1.15 (fast, punchy)
- `[sighing]` → pace 0.75 (slow, heavy)
- `[resigned]` → pace 0.70 (slow, trailing)
- `[whispering]` → pace 0.65 (slowest, intimate)
